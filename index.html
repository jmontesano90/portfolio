Gaea: Ecological Simulator

Gaea is an ecological simulator that simulates three different species of plants. Watch them grow, compete, and reproduce in real-time! It can be used to help understand the basics of evolution, natural selection, and Mendelian genetics!
Uses React, Node, Express, HTML, CSS, JS, JQuery, PostgreSQL

Transect Generator

Geb is a transect generator aimed at ecologists, biologists, entomologists, or anyone that uses transect based sampling! Use Geb to specify template dimensions, then allow it to tell you where to sample using complete random transect generation.
Uses React, Node, Express, HTML, CSS, JS, JQuery, PostgreSQL

Dark Rolls

Dark Rolls is a dice simulator for the Dark Souls board game. Its function is to give you useful statistics on gears dice rolls so players can make better-informed decisions. It can also be used to get the probability of an attack, which can be useful in a pinch!
Uses HTML, CSS, JS and JQuery
