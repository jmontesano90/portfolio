Dark Rolls

Dark Rolls is a dice simulator for the Dark Souls board game. It's function is to give you useful statitistics on gears dice rolls so players can make better informed decisions. It can also be useful to get the probability of an attack, which can be useful in a pinch!
Uses HTML, CSS, JS and JQuery
